Descripción

Este applet está basado en el applet Lista de ventanas predeterminado de Cinnamon.

Diferencias con el original

  • Se agregó la opción para eliminar la etiqueta de los botones de lista de ventanas.
  • Se agregó la opción para eliminar las sugerencias de los botones de lista de ventanas.
  • Iconos añadidos al menú contextual del botón de lista de ventanas.
  • Se agregó la opción para invertir los elementos del menú contextual.
  • Se agregó la opción para ocultar/mover el submenú Preferencias.

  • Si este xlet se instaló manualmente y no a través de Configuración de Cinnamon, las localizaciones se pueden instalar ejecutando el archivo llamado helper.py desde una terminal abierta dentro de la carpeta del xlet.

Help for Window list (Fork By Odyseus)

IMPORTANT!!!

Never delete any of the files found inside this xlet folder. It might break this xlet functionality.

Bug reports, feature requests and contributions should be done on this xlet's repository linked next. GitLab

Compatibility

Cinnamon3.0 Cinnamon3.2 Cinnamon3.4 Cinnamon3.6 Cinnamon3.8 Cinnamon4.0 Cinnamon4.2 Cinnamon4.4 Cinnamon4.6 Cinnamon4.8 Cinnamon5.0 Cinnamon5.2 Cinnamon5.4 Cinnamon5.6 Cinnamon5.8 Cinnamon6.0

Do not install on any other version of Cinnamon.

Description

This applet is a fork of the default Window list applet shipped with Cinnamon.

Differences with the original applet

  • Added option to remove the label from the window list buttons.
  • Added option to remove the tooltips from the window list buttons.
  • Added icons to the window list button's context menu.
  • Added option to invert the contex menu items.
  • Added option to hide/move the Preferences sub menu.

Inverted menu on top panel

Applets/Desklets/Extensions (a.k.a. xlets) localization

  • If this xlet was installed from Cinnamon Settings, all of this xlet's localizations were automatically installed.
  • If this xlet was installed manually and not trough Cinnamon Settings, localizations can be installed by executing the script called helper.py from a terminal opened inside the xlet's folder.
The following sections are available only in English.
