python_modules.python_utils.multi_select module¶

Multi selection menu creator.

class python_modules.python_utils.multi_select.MultiSelect(menu_items=[], title='', arrow='==>', footer='Space = toggle ─ Enter = accept ─ q = cancel', more='...', char_selected='[X]', char_empty='[ ]')[source]¶

Bases: object

Allows you to select from a list with curses.

Based on picker Python module.

Example

opts = MultiSelect(
    title='Select files to delete',
    options=[
        ".autofsck ", ".autorelabel", "bin/", "boot/",
        "cgroup/", "dev/", "etc/", "home/", "installimage.conf",
    ]
).getSelected()

if not opts:
    print("Aborted!")
else:
    print(opts)

Initialization.
