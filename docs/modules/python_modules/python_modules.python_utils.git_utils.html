python_modules.python_utils.git_utils.manage_repo(mechanism, action, subtrees=[], cwd=None, logger=None)[source]¶

Manage repository.

Perform some complex tasks on a repository. Mostly sub-trees and sub-modules initialization.

Parameters

  • mechanism (str) – Which “mechanism” to work with (“submodule” or “subtree”).

  • action (str) – Which action to perform (“init” or “update”).

  • subtrees (list, optional) –

    A list of dictionaries representing sub-tree options.

    • remote_name: The sub-tree remote name.

    • remote_url: The sub-tree remote URL.

    • path: Path inside the parent repository where the sub-tree will be stored.

  • cwd (None, optional) – Path to working directory. It should be a folder that belongs to a Git repository.

  • logger (object) – See <class LogSystem>.
