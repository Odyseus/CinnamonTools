If this xlet was installed from Cinnamon Settings, all of this xlet's localizations were automatically installed.

Description

Color Blindness Assistant is a Cinnamon extension that provides tools to assist users with color vision deficiency (CVD).

Features

This extension provides mainly of three features.

  • Color blindness compensation: A user can apply an effect (specific to her/his pathology) that will (might) help her/him to better differentiate colors.
  • Color blindness simulation: A developer can apply an effect that will show her/him how a person with color vision deficiency will (might) see certain color combinations.
  • Color naming: This feature is useful for knowing the name of the color that's immediately under the cursor.

The following sections are available only in English.

This change log is only valid for the version of the xlet hosted on its original repository

  • Date: Tue, 16 Apr 2019 14:19:55 -0300
  • Commit: 56e7d49
  • Author: Odyseus
Initial commit. An extension whose main purpose is to serve as an assistant to users with color vision deficiency (CVD). It provides compensation effects (it allows users with CVD to better differentiate colors that they can't see), simulation effects (it allows developers to see how users with CVD see certain combination of colors and ascertain if a color palette is *CVD friendly*) and a *color naming tool* that allows to get the name of a color on the screen.
